# Search Session Logs

This directory contains log files from job search sessions.

Each file is named: `search_session_YYYYMMDD_HHMMSS.log`

Log files track:
- Search configuration (score threshold, max results)
- Active job role used
- URLs searched from watchlist
- Opportunities found and saved
- Session duration and statistics
- Any errors encountered

These logs are referenced in the search_sessions database table.
